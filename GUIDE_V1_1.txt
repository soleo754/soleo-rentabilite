SOLÉO PILOT — RENTABILITÉ LAB V1.1 — VERSION FINALE AVEC IMPORT JSON

MISE À JOUR
1. Exporter une sauvegarde JSON depuis l'application actuelle si elle contient encore des données.
2. Remplacer sur GitHub les fichiers index.html, _headers et GUIDE_V1_1.txt.
3. Faire Commit changes. Cloudflare redéploie automatiquement.
4. Ouvrir l'application et actualiser avec Ctrl+F5.
5. Aller dans Données LAB puis cliquer sur « Importer une sauvegarde JSON ».
6. Choisir le fichier de sauvegarde .json et confirmer le remplacement des données locales.
